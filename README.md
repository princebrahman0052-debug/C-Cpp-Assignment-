## What is a variable in C/C++?

A variable in C/C++ is a named storage location in memory that stores a value. The value of a variable can change during program execution.

Example:
int age = 20;

## Why do we use variables in programs?

Variables are used in programs to store and manage data. They allow a program to save values, use them in calculations, and change them whenever needed during execution. Using variables makes programs flexible, reusable, and easier to understand.

Example:
int age = 20;
age = 21;  // The value of the variable can be changed
